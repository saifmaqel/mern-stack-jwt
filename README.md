#  Workout Tracker (MERN + JWT + Rate Limiting)

A simple **MERN stack** project for managing workouts.  
It supports **JWT authentication** for secure access and **rate limiting** to prevent abuse of the API.

---

##  Tech Stack

**Frontend**
- React + Vite + TypeScript
- React Query (data fetching & caching)

**Backend**
- Node.js + Express
- MongoDB + Mongoose
- JSON Web Tokens (JWT) for auth
- Express-rate-limit for API throttling

---
